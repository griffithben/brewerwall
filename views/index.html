Simple Requests

Perform a GET request to retrieve all yeasts

$ curl http://brewerwall.com/api/yeasts

Perform a GET request on a specific ID

$ curl http://brewerwall.com/api/yeasts/1

Perform a POST request to filter on certain parameters

$ curl --data "temperature=80" http://brewerwall.dev/api/yeasts/
